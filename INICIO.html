RESUMEN DEL PROYECTO

Proyecto de una cerradura digital para caja fuerte de uso personal, con diseño inspirado en el "Ender Chest". Combina tres métodos de autenticación: huella dactilar, tarjeta RFID y código numérico, buscando ofrecer una solución segura y visualmente atractiva.

Incluye una página web informativa y una aplicación Android para control remoto. La app permitirá apertura remota, registro y reportes de accesos, cambio de clave y notificaciones de alarma. También se prevé una sección de compra con inicio de sesión y almacenamiento en base de datos para usuarios y pedidos.

Componentes clave: microcontrolador (por ejemplo Arduino) que procesa las entradas, sensores biométricos (huella), lector RFID, teclado para código numérico y un actuador/mecanismo de cierre. La funcionalidad básica sigue tres etapas:

  1. Entrada: el usuario presenta credenciales (huella/RFID/código).
  2. Procesado: el microcontrolador verifica la autenticidad contra la base de datos o credenciales almacenadas.
  3. Cierre/Apertura: si la autenticación es válida, se activa el mecanismo que abre la cerradura; si no, se deniega el acceso y puede registrarse el intento.

En resumen: diseño estético + autenticación triple + control remoto y registro; pensado para seguridad personal con posibilidad de comercialización y expansión a variantes visuales.
