Componentes del Proyecto

    Arduino UNO

    Placa de desarrollo con múltiples pines digitales y analógicos, que actúa como el cerebro del sistema.

    Micro Servo SG90

    Servo motor que necesita 5V para funcionar, ideal para movimientos precisos.

    Display LCD 16x2 con Módulo I2C

    Muestra información al usuario de forma clara y sencilla. Se monta detrás de la pantalla LCD para comunicación con menos cables.

    Keypad 4x4

    Teclado matricial de 16 teclas para ingreso de datos.

    Sensor de Huella ZA608

    Lee huellas dactilares para autenticación segura.

    Lector de Tarjetas RC522

    Detecta tarjetas RFID para control de acceso sin contacto.

    Botón Interruptor

    Permite encender o apagar todo el sistema fácilmente.

    Protoboard y Cables Dupont

    Para conexiones seguras y ordenadas sin necesidad de soldar.

    MODULO ESP-01

    Módulo Wi-Fi ESP-01 para comunicación inalámbrica y control remoto.

    Módulo Bluetooth HC-06

    Permite comunicación inalámbrica con otros dispositivos.

    Buzzer Activo

    Emite sonido al recibir señal digital alta.

    Cable Alimentacion

    lleva la corriente desde un adaptador de pared hasta el dispositivo; suministra un voltaje DC fijo (ej. 9V o 12V) para que el equipo funcione

    Cable Arduino

    sirve para alimentar la placa y, sobre todo, enviar/programar el código desde la PC y para comunicación serial.

    Modulo PCF

    permite agregar más entradas y salidas digitales a tu Arduino.

    Modulo I2C

    protocolo de comunicación usado para conectar varios sensores y módulos

    Buzzer Pasivo

    Permite reproducir tonos y melodías mediante señales de frecuencia.
